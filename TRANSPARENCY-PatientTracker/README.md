TRANSPARENCY-PatientTracker

Framework. The project is currently in its foundational development phase, focusing on proper system 
setup, virtual environment configuration and version control.
A minimal custom user model has been implemented to support hospital staff roles and departmental 
assignments while preserving Djangoâ€™s default authentication and administrative features.
Core applications for user management, patient records and visit tracking have been created, with
